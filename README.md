# 💰 Finance Tracker — Cyberpunk Edition

A sleek Cyberpunk-styled personal finance management app built with HTML, CSS, and JavaScript.

## 🚀 Live Demo
[View Live App](https://starlit-trifle-84802b.netlify.app/finance-tracker.html)

## ✨ Features
- Income and expense tracking
- Budget categories (Housing, Food, Transport, Entertainment, Healthcare, Shopping, Salary)
- Monthly income vs expenses bar charts
- Category breakdown with progress bars
- Monthly summary table
- Local storage — data saves automatically
- Filter transactions by month
- Cyberpunk neon design with glitch effects

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom Cyberpunk theme)
- JavaScript (Vanilla)
- Local Storage API

## 📱 How To Use
1. Add income or expenses using the form
2. Select category and date
3. View charts and breakdowns in the Charts and Categories tabs
4. Data saves automatically to your browser
